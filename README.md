
This project is a C++ implementation of a Doubly Linked List. In a normal linked list, each node points to the node ahead of it. 
In a Doubly linked list, each node points to the node in front of it and the node behind it. Working on this project resulted in 
several memory issues that I eventually fixed, along with edge cases that needed to be considered. Overall, I'd say it gave me a
far better understanding of Data Structures and how they work under the hood.
